# Valentine's Day Tribute - Friendship Edition

A modern, cinematic web experience celebrating friendship, built with React, Tailwind CSS, and GSAP.

## Features

- **Immersive Hero Section**: Parallax background with deep red tones and soft gradients.
- **Horizontal Scrolling Gallery**: GSAP ScrollTrigger-powered memories timeline.
- **Emotive Letter**: Animated text reveal for a heartfelt message.
- **Smooth Interaction**: Lenis scroll smoothing for a premium feel.
- **Responsive Design**: Optimized for all devices.

## Tech Stack

- React 19
- Vite
- Tailwind CSS v4
- GSAP (GreenSock Animation Platform)
- Lenis (Smooth Scrolling)

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open `http://localhost:5173` in your browser.

## Customization

- **Images**: Update image URLs in `src/components/Gallery.jsx` and `src/components/Hero.jsx`.
- **Text**: Edit the content in `src/components/Letter.jsx`.
- **Colors**: Modified via Tailwind classes (rose-50 to rose-950).
